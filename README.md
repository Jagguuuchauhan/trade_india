## 📖 Overview

**Trade India** is a modern **Forex Virtual Trading Platform** developed using **HTML, CSS, JavaScript, Firebase Authentication, and TradingView Widgets**.

This platform allows users to practice Forex trading with virtual money in a realistic environment. It includes live TradingView charts, market overview, virtual portfolio management, educational content, and secure authentication.

> ⚠️ **This project is built for educational and learning purposes only. No real money is involved.**

# 🔥 Firebase Setup

1. Create a Firebase Project.
2. Enable Authentication.
3. Enable Email/Password Login.
4. Enable Google Login.
5. Enable Facebook Login.
6. Enable GitHub Login.
7. Enable Realtime Database.
8. Replace the Firebase configuration inside **login.js**.

```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "...",
  databaseURL: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```
